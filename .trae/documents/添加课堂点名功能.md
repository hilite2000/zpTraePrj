# 课堂点名功能实现计划

## 功能需求

创建一个课堂点名系统，允许教师通过网页界面进行随机点名，具体功能包括：

1. **学生名单配置**：在单独的配置文件中存储学生姓名
2. **随机点名**：点击开始按钮后快速随机显示学生姓名
3. **停止功能**：点击停止按钮后显示当前选中的学生姓名
4. **美观界面**：符合网站整体设计风格

## 实现方案

### 1. 创建新文件

1. **`rollcall.html`**：点名功能主页面
2. **`students.json`**：学生姓名配置文件

### 2. 技术实现

* **HTML结构**：创建响应式布局，包含标题、学生显示区域、控制按钮

* **CSS样式**：使用Tailwind CSS，保持与网站整体风格一致

* **JavaScript功能**：

  * 读取JSON配置文件获取学生名单

  * 实现随机点名算法

  * 添加开始/停止控制逻辑

  * 实现动画效果

### 3. 页面设计

* **顶部导航**：与其他页面保持一致

* **标题区域**：显示"课堂点名"标题

* **姓名显示区域**：大屏幕显示当前选中的学生姓名

* **控制按钮**：开始/停止按钮，带有动画效果

* **学生列表**：显示当前班级的学生名单

### 4. 功能流程

1. 页面加载时读取`students.json`文件
2. 点击"开始"按钮，开始随机切换显示学生姓名
3. 点击"停止"按钮，停止随机切换并显示最终选中的学生姓名
4. 可以通过修改`students.json`文件来更新学生名单

### 5. 界面设计

* 使用与网站一致的蓝色系配色

* 添加适当的动画效果，增强用户体验

* 确保响应式设计，在不同设备上都能正常显示

* 添加视觉反馈，如按钮状态变化、姓名切换动画等
